# 学生管理系统简介 python213

## 项目概述

学生管理系统是基于Python语言的Flask框架开发的Web应用，旨在为初学者提供一个入门级的开发教程。该系统涵盖了前端展示与后端逻辑处理，使用MySQL数据库进行数据存储。

## 技术栈

- 前端：HTML + CSS
- 后端：Python
- 数据库：MySQL

## 功能模块

系统主要包括以下功能模块：

- 学生信息管理
- 课程信息管理
- 成绩管理
- 用户权限控制

## 系统角色

- 管理员：负责系统维护与用户管理
- 教师：负责成绩录入与课程管理
- 学生：查询个人信息与成绩

## 运行环境

- Python环境
- Flask框架
- MySQL数据库

## 部署步骤

1. 安装Python环境
2. 安装Flask框架
3. 配置MySQL数据库
4. 导入项目中的SQL文件以初始化数据库
5. 运行Python应用

## 目录结构

```
/
├── app                # Flask应用目录
│   ├── models         # 数据模型
│   ├── views          # 视图逻辑
│   ├── templates      # 前端模板
│   └── static         # 静态资源
├── venv               # Python虚拟环境
├── requirements.txt   # 项目依赖
└── config.py          # 配置文件
```
